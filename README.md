# Rubrika
A Java-based personal knowledge management system for tracking hobby progress. Designed as both an educational project and practical tool for PKM.

## ✨ Features

### Planned
- Obsidian-style [[wiki links]]
- Knowledge graph visualization
- Markdown rendering & themes
- Spell checking integration
- **Daily Note Generation**: One-click creation of timestamped notes
- **Topic Organization**: Hierarchical structure (User → Topic → Note)
- **Search & Navigation**: Fuzzy search with date/topic filtering
- **Data Export**: CSV generation for analysis
- **Sorting**: Automatic topic sorting by title

## 🛠 Tech Stack

**Backend**
- Java, Spring Boot
- PostgreSQL, Hibernate
- JUnit, Mockito

**Future Plans**
- Spring Security
- Dockerize everything
- Kafka event streaming
- CI/CD pipelines
